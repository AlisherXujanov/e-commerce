## To push into github
<!-- STEPS -->


<!-- Create a branch and checkout into it -->
- git checkout -b 'branch-name'

<!-- To return back -->
- git checkout main  
- git checkout branch-name   <!-- Go to the branch -->

<!-- send into github -->
1. git add .
2. git commit -m "message"
3. git push
